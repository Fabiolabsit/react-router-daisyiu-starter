## This is the starter template for the project that requires

- React
- React Router
- Tailwind CSS
- DaisyUI
- React Icons

## How to use

step 1: Download the template or Clone the repo

```bash
git clone
```

step 2: Install all the dependencies

```bash
yarn
```

or

```bash
npm install
```

step 3: Start the development server

```bash
yarn start
```

or

```bash
npm start
```

step 4: Build the project

```bash
yarn build
```

or

```bash
npm run build
```
